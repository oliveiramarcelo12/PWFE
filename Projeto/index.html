Lâmpada Interativa

Este projeto é uma página da web interativa que simula o controle de uma lâmpada. Os principais elementos da página incluem uma imagem de uma lâmpada, um botão de alternância para ligar/desligar a lâmpada e um título "Interativo". O objetivo é permitir que os usuários cliquem no botão para controlar o estado da lâmpada.

Cronômetro

Esta página HTML apresenta um cronômetro interativo com botões para iniciar, parar e resetar. Ela inclui uma imagem representativa do cronômetro e um menu de navegação que redireciona para a página inicial. Os botões permitem controlar o cronômetro, enquanto os elementos de exibição de tempo estão formatados em blocos separados para horas, minutos, segundos e milissegundos. O arquivo JavaScript "cronometro.js" é vinculado para adicionar funcionalidade ao cronômetro.

Jogo da #

Esta página HTML implementa o clássico jogo da velha para dois jogadores, onde "X" e "O" competem para completar uma linha, coluna ou diagonal com os seus respectivos símbolos. Os jogadores alternam entre si e o jogo verifica automaticamente se alguém ganhou ou se houve um empate após cada jogada. A pontuação de vitórias para os jogadores "X" e "O" é exibida na parte inferior da página para acompanhar o desempenho. O jogo utiliza HTML, CSS e JavaScript para criar uma experiência interativa e envolvente para os jogadores. As animações de vitória e empate adicionam um toque de estilo à página, enquanto os botões clicáveis permitem que os jogadores façam suas jogadas. Divirta-se jogando este clássico jogo da velha!.
